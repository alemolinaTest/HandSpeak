It's a simple Jetpack Compose implementation for
Google mediapipe Gestur Recognizer Implementation
Camera 2 Implementation

Just stand in front of the device camera and try these gestures with one hand

             Thumb_Up" -> "👍"

            "Closed_Fist" -> "✊"
            "Victory" -> "✌️"
